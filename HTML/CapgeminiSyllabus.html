Capgemini Exceller Syllabus 2023

Capgemini Syllabus For Placement

Capgemini Exceller has changed its Syllabus for its recruitment drive in march this year. They have changed the number of rounds and the entire selection process. The first round will be Technical Test round which will be conducted by CoCubes/ AMCAT, followed by English Test, Game based Aptitude etc for Capgemini Exceller Recruitment Drive. Below we have mentioned all the details please go through all of them


Capgemini Exceller Syllabus For 2023


There will be six sections in the Online Written Test. The Six sections of Capgemini Exceller are:-
  1. Technical Test
  2. English Communication Test
  3. Aptitude Test(Game Based)
  4. Behavioral Competency Profiling
  5. Spoken English Test
  6. Coding
  7. Technical Interview
  8. HR Interview

Note - Capgemini Exceller also asks coding questions for a different profile, wherein you have to solve 3 questions within 75 mins

Capgemini Exceller Behavioral Competency


This is another new section in the exam. Don't get confused with the name of this section, this is just another name for the Psychometric Test, which many of you may have given before in some other drive of companies like IBM, etc. Now, please don't get cheered up, as this may be one of the most difficult sections that you may come across. Since there are no correct and wrong answers in this section, whatever option you choose, defines you as a person. So you need to be very careful while answering these questions



Capgemini Exceller English Communication Test

This will be an MCQ based test, where there will be 4 options for every question, and you have to choose the correct answer among them. This test will be testing your grammatical and vocabulary skills. Some of the topics that are going to be asked in this exam are -


  • Sentence Correction
  • Prepositions
  • Grammar
  • Reading Comprehension
  • Synonyms & Antonym
  • Speech and Tenses
  • Article
  • Sentence Selection
  • Spotting Error
  • Sentence Arrangement


Capgemini Exceller Syllabus for Technical Test

The following topics are asked in Exam for the freshers in Capgemini Exceller Technical Test.


  • C
  • C++
  • OOPS
  • Data Structures
  • Pseudo Code

This is an elimination round, so that means that for sitting in the next round you first have to clear this round. If you fail to do so, then you will not be allowed to proceed further in the process

Capgemini Exceller Game Based Aptitude


This section is there to test your multitasking and decision-making skills. There will be a total of 24 games, out of which the system will select 4 games for you. Your task is to solve those 4 games accurately and speedily, without making as few mistakes as you can


  • Deductive Logical Thinking(Geo-Sudo)
  • Inductive-logical Thinking
  • Grid Challenge
  • Motion Challenge
  • Switch Challenge
  • Digit Challenge
